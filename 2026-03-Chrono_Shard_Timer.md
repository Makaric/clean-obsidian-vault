# 🌌 Chrono-Shard (Focus Timer)

## 🎯 Суть (Core Essence)
> *Что мы создаём из Пустоты? Какая главная ценность для юзеров (кочевников)?*
- Инструмент для глубокого фокуса. Помодоро-таймер, который помогает отсечь шум Пустоши и погрузиться в поток творения или медитации.

## 🛠 Механика (Architecture)
> *Mad Max эстетика, стек технологий, ключевые фичи.*
- Vanilla JS (`setInterval` для логики таймера).
- CSS для визуализации: ржавые текстуры, эффект свечения вакуумных ламп (Nixie tubes), тяжелые анимации переключателей.
- `AudioContext` для процедурных звуков: скрежет металла при старте, вой сирены по окончании.
- Настраиваемые интервалы (25/5, 50/10).
