# Clean Obsidian Vault

A ready-to-use Obsidian vault structure based on the PARA method, tuned for digital nomads and knowledge craftsmen.

## Structure

```
vault/
  00_INBOX/       - capture everything here first
  10_PROJECTS/    - active projects with deadlines
  20_AREAS/       - ongoing responsibilities
  30_RESOURCES/   - reference material & topics of interest
  40_FINANCE/     - wallets, budgets, crypto notes
  99_ARCHIVE/     - completed & inactive items
  Templates/      - reusable note templates
```

## Quick Start

1. Download this repo as ZIP (Code -> Download ZIP)
2. Unzip anywhere on your drive
3. Open Obsidian -> "Open folder as vault" -> select the `vault/` folder
4. Start writing. Inbox first, organize later.

## Philosophy

- **Inbox zero**: everything enters through `00_INBOX`, gets sorted later
- **PARA**: Projects / Areas / Resources / Archive — simple, scalable
- **Templates**: use `Project_Manifest.md` to kickstart any new project
- **No plugins required**: works with vanilla Obsidian out of the box
